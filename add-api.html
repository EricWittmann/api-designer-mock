Add an Existing API

What is this page?

Here you can add an existing API to the Studio by telling us where the API's OpenAPI document is located. Simply provide the GitHub URL of your OpenAPI document and we'll do the rest!

What do I do?

Simply copy/paste (or drag and drop!) the GitHub URL of your existing OpenAPI document from your browser into the form and then click "Add API". Easy peasy, lemon squeezy!
