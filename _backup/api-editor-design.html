Get a Published API

Returns information about a single published API. Required parameters include the organization id, API id, and API version.

Parameters

Parameter Description Type
organizationId The API's organization id. string
apiId The API's unique identifier. string
version The version of the API to get. string

Response (Success)

A successful call will result in a 200 OK HTTP response.

Response Body: Api
Response Content Type:

Response (Failure)

Status Code Reason Type
403 User was not authorized to perform action. NotAuthorizedError

Retire (Delete) an API

Retires a previously published API from the Gateway, immediately making the API unavailable for consumption.

Parameters

Parameter Description Type
organizationId The API's organization id. string
apiId The API's unique identifier. string
version The version of the API to get. string

Response (Success)

A successful call will result in a 204 No Content HTTP response.

Response (Failure)

Status Code Reason Type
400 Unexpected error when attempting the delete. ApiDeleteError
403 User was not authorized to perform action. NotAuthorizedError
